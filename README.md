Basic 3D Engine (OpenGL)

This project is a simple 3D engine created for educational purposes using OpenGL. Its main goal is to explore and understand the fundamentals of real-time rendering, graphics pipelines, and interactive 3D environments.

The engine includes a custom-written shader system, allowing full control over vertex and fragment processing. It also implements core logic for generating, displaying, and dynamically adding 3D objects (such as cubes) within the scene.

Additionally, the project features basic camera movement and input handling, enabling navigation through the 3D space.
